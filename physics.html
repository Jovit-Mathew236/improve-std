Errors

➤
  • percentage error δa = Δamean/amean×100%
    1. Δamean=Absolute error
    2. amean=Relative error

➤ Combination of error

  • Z=A+B
    The maximum possible error in Z is given by
    ΔZ=ΔA+ΔB
  • Z=A-B
    The maximum possible error in Z is given by
    ΔZ=ΔA+ΔB
  • Z=AB
    Maximum relative error in Z is given by
    ΔZ/Z=ΔA/A+ΔB/B
    Maximum percentage error in Z is given by
    ΔZ/Z×100=ΔA/A×100 + ΔB/B×100
  • Z=A/B
    Maximum relative error in Z is given by
    ΔZ/Z=ΔA/A+ΔB/B
    Maximum percentage error in Z is given by
    ΔZ/Z×100=ΔA/A×100+ΔB/B×100
  • Z=AmMaximum relative error in Z is given by
    ΔZ/Z=m×ΔA/A
  • Z=ApBq/Cr
    Maximum relative error in Z is given by
    ΔZ/Z= p×ΔA/A + q×ΔB/B + r×ΔC/C
